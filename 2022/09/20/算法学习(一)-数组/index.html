算法学习(一)-数组

一、数组理论基础

​ 在学习数组理论之前，我们要明确一个概念：数组内存空间的地址是连续的。也就是说在进行数组删除等操作时，我们不能够删除原来的元素，而是对原来的元素进行覆盖！当然这是一维数组的地址分布，那么二维数组又是怎样分布的呢？其实不同编程语言的内存管理是不一样的，以C++为例，在C++中二维数组是连续分布的，而在java中二维数组的每一行头结点的地址是没有规则的，更谈不上连续。

二、二分查找（704）

leetcode题目链接：https://leetcode.cn/problems/binary-search/

题目描述：

​ 给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target ，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

​ 二分查找是数组问题中非常经典的一个问题，其难点在于两个边界的确定：1.while循环中left<right还是left<=right?，2.当中间值>目标值时，right的取值是mid还是mid-1？其实这个问题通过数学的期间就可以获得答案，当我们使用左闭右闭区间时，left是可以等于right的，所以我们while循环中选用left<=right；而当中间值>目标值时，此时我们可以知道mid值以及不符合与目标值相等这个条件了，所以我们舍弃这个值，选用mid-1。而对于左闭右开区间同理即可得到另一组取值。

四、有序数组的平方（977）

leetcode题目链接：https://leetcode.cn/problems/squares-of-a-sorted-array/

题目描述：

​ 给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

示例：

​ 当数组全是正数时，这道题非常好解，但是我们必须考虑负数，观察我们发现，平方的最大值要么在数组最前面，要么在数组最后面，针对这个规律，我们可以选用双指针法，head从开始，tail从尾开始，当head^2>=tail^2时，将head^2存入新数组尾部，head++，反之将tail^2存入新数组尾部，tail++，直到head>tail。

五、长度最小的子数组（209）

leetcode题目链接：https://leetcode.cn/problems/minimum-size-subarray-sum/

题目描述：

​ 给定一个含有 n 个正整数的数组和一个正整数 target 。

找出该数组中满足其和 ≥ target 的长度最小的 连续子数组 [numsl, numsl+1, …, numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。

示例：

​ 本题可以通过滑动窗口法来解决。简单来说就是确定两个i、j指针，j从起点开始前进，当i、j两指针之间的数的和大于等于目标值时，记录此时的长度，移动i，每次i、j两指针间的数的和大于等于七时，就记录下此时的长度，最终比较选出最小长度。
